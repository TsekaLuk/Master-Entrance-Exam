# 手绘快题技法

> 每周至少一次限时手绘训练（30分钟）

## 界面手绘要素

### 基本组件
- [ ] 导航栏（顶部/底部 Tab）
- [ ] 按钮（主要/次要/文字按钮）
- [ ] 卡片（Card）
- [ ] 列表项（List Item）
- [ ] 输入框（Input Field）
- [ ] 图标（Icon）— 简化几何形状

### 布局原则
- **网格系统**：8px 基础单位
- **留白**：内容区域与边缘的间距
- **对齐**：左对齐为主，保持一致性

## 快速草图流程（30分钟内）

```
5分钟：理解题目，确定核心功能
5分钟：确定信息架构（页面列表）
10分钟：关键页面线框图（2-3个屏幕）
5分钟：标注说明（功能描述）
5分钟：整体检查与收尾
```
