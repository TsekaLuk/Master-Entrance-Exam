## ⌨️ 快捷键的使用（键盘党最爱）

- 下面是列出的常用命令，以及快捷键。建议学会，可以大大的提高使用效率。
- 由于我没有 Windows 系统电脑，快捷键是按照 Mac 系统设计的，如果你发现快捷键无效，可以去自行设置。设置方法：点击设置打开设置页面，找到快捷键栏目，进去就可以自行设置
	- ![[Pasted image 20240704183537.png]]
- Cmd 对应 Windows 系统中的 Ctrl 键，Opt 对应的是 Windows 中的 Alt 键，shift 键一样。括号里代表着对应的插件

| 💻 **常用命令**               | ⌨️ **快捷键**  |
| ------------------------- | ----------- |
| 打开命令行                     | Cmd+O       |
| 打开快速添加                    | Cmd+N       |
| 切换阅读/编辑视图                 | Cmd+E       |
| 打开设置                      | Cmd+，       |
| 全局搜索                      | Cmd+F       |
| 切换/创建文件                   | Cmd+P       |
| 保存当前文件                    | Cmd+S       |
| 插入链接                      | Cmd+K       |
| 返回                        | Cmd+【       |
| 前进                        | Cmd+】       |
| 加载工作区布局                   | Opt+Tab     |
| 关闭当前标签页                   | Cmd+W       |
| 加粗/取消加粗                   | Cmd+B       |
| 倾斜/取消倾斜                   | Cmd+I       |
| ==切换列表/待办事项==             | ==Cmd+L==   |
| ==打开/创建今天的日记==            | ==Cmd+J==   |
| ==打开主页==                  | ==Cmd+h==   |
| ==打开时间线==                 | ==Cmd+G==   |
| 转化成/编辑选中的任务               | Cmd+；       |
| 删除段落                      | Cmd+D       |
| 新标签页                      | Cmd+T       |
| 新建笔记                      | Cmd+shift+N |
| 与上一行互换                    | Cmd+opt+↑   |
| 与下一行互换                    | Cmd+opt+↓   |
| 将列表（任务）及子列表向上移动           | Cmd+shift+↑ |
| 将列表（任务）及子列表向下移动           | Cmd+shift+↓ |
| 在当前笔记中查找并替换               | Cmd+opt+F   |

| 折叠/展开右侧边栏                 | Opt+2       |

| 深度聚焦（Zoom in）             | Cmd+.       |
| 退出深度聚焦模式（Zoom Out）        | Cmd+shift+. |

| 转到下一个标签页（Outliner）        | Cmd+shift+】 |
| 随机漫步（Smart Random Note）   | Cmd+R       |
| 高级随机漫步（Smart Random Note） | Cmd+shift+R |
| 插入带颜色的时间戳                 | Opt+T       |
| 插入当前时间                    | Cmd+shift+T |
