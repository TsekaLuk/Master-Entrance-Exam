# Midjourney 产品案例分析

> 843 设计方向高频案例，重点掌握"AI 辅助创意设计"和"人机协同创作"

---

## 基本信息

| 项目 | 内容 |
|------|------|
| 产品定位 | AI 图像生成工具 |
| 所属公司 | Midjourney Inc.（独立实验室）|
| 上线时间 | 2022 年 7 月（公测）|
| 核心技术 | 扩散模型（Diffusion Model）|
| 商业模式 | 纯订阅制（无免费版）|
| 用户规模 | 约 1600 万付费用户（2024）|

---

## 四维分析

### 1. 洞察力 — 发现了什么问题？

**设计师痛点：**
- 创意概念草图耗时长（手绘或找素材）
- 灵感枯竭时缺乏外部刺激
- 非专业用户无法将脑中创意变成可视化图像

**市场洞察：**
- 视觉内容需求爆炸式增长（社交媒体、品牌设计、游戏概念图）
- 文字→图像的转换是长期未被满足的"创意民主化"需求
- 设计师不会被 AI 替代，但会被"使用 AI 的设计师"替代

---

### 2. 思维力 — 如何解决问题？

**设计思维应用：**

| 阶段 | Midjourney 的做法 |
|------|-----------------|
| 同理心 | 观察设计师和创意人的工作流，理解"从概念到视觉"的痛点 |
| 定义问题 | 让普通人通过文字描述生成高质量图像 |
| 创意发散 | 选择 Discord 作为分发平台（无需自建 App，直接触达创意社区）|
| 原型验证 | 公测阶段让用户共同探索 Prompt 写法，形成使用规范 |
| 迭代测试 | V1→V6 每代模型提升真实感、细节度和指令理解力 |

**核心交互设计决策：**
- **Discord 分发**：借助现有社区平台（无需自建），冷启动成本极低
- **公开生成**：所有用户图像默认可见，形成"互相学习 Prompt"的社区效应
- **图像变体（Variations）**：一键生成 4 张变体，降低"选择哪张"的决策成本
- **Upscale 放大**：二阶段工作流（先构图 → 再精修），符合设计师思维

**互联网思维应用：**
- 社区驱动：Discord 频道中用户互相学习，UGC 内容降低平台教育成本
- 数据飞轮：用户的图像选择（哪张更好）反馈训练模型
- 迭代快：每 3-6 个月发布新版本模型，能力大幅跃升

---

### 3. 学习力 — 跨界融合了什么？

| 借鉴来源 | 融合方式 |
|---------|---------|
| 游戏社区（Discord）| 借用游戏玩家的社区文化，AIGC 创作变成"玩" |
| 设计工具（Adobe）| 工作流参考（草图→精修→导出），但门槛大幅降低 |
| 社交媒体 | 公开图库形成作品展示激励，用户分享传播 |
| 摄影构图学 | Prompt 工程融合专业术语（光影/景深/风格），提升输出质量 |

---

### 4. 表现力 — 可视化草图

```
[ 用户输入 Prompt（文字描述）]
         ↓
[ 扩散模型（Diffusion Model）]
         ↓
  从噪声逐步生成清晰图像（约 60 步去噪）
         ↓
[ 输出 4 张候选图（2×2 网格）]
         ↓
用户选择：[U1-U4 放大] / [V1-V4 变体]
         ↓
[ 高清最终图 → 下载/分享 ]
```

**Prompt 结构示意：**
```
主体描述 + 风格关键词 + 技术参数
例：A futuristic smartphone interface, clean UI design,
    soft blue gradient, minimalist, --ar 9:16 --v 6
```

## 对 843 设计题的启示

> 考题如果问"AIGC 如何改变设计行业"，可从以下角度展开：

1. **角色转变**：设计师从"执行者"→"创意导演"（人写 Prompt，AI 执行）
2. **效率革命**：概念阶段从"数天"→"数小时"
3. **门槛重塑**：非科班用户也能产出专业视觉，设计民主化
4. **新技能需求**：Prompt 工程成为设计师必备技能
