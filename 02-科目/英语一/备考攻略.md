# 考研英语一备考攻略

## 目标：75分

75分是性价比最高的分数，80+需要额外大量时间，不如把时间花在专业课上。

## 各题型目标分值

| 题型 | 分值 | 目标得分 | 说明 |
|------|------|----------|------|
| 完形填空 | 10 | 6 | |
| 阅读理解 | 40 | 32 | 最重要 |
| 新题型 | 10 | 8 | |
| 翻译 | 10 | 5 | |
| 小作文 | 10 | 7 | |
| 大作文 | 20 | 13 | |

## 复习规划

### 基础阶段（3-6月）
- 单词：考研词汇过2-3遍
- 长难句：每天分析2-3句
- 早读：培养语感

### 强化阶段（7-9月）
- 真题阅读：2000-2020年阅读真题精做
- 每篇阅读精读：单词、长难句、选项分析
- 预留近2年真题模拟

### 冲刺阶段（10-12月）
- 真题二刷、三刷
- 作文模板整理
- 全真模拟

## 关键提醒

1. **紧抓真题** - 不要做预测题，真题才是最重要的
2. **阅读为王** - 阅读占40分，是决定性题型
3. **作文模板** - 整理自己的模板，不要完全照搬
4. **模拟训练** - 预留真题考前模拟，训练3小时节奏
