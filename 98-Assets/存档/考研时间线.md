## 3. 分阶段执行主线

### 3.1 基础期（2/16-6/30）

- 数学二：函数极限连续 → 导数微分 → 积分与应用 → 微分方程 → 多元函数 → 二重积分 → 线代（行列式/矩阵/向量/方程组/特征值/二次型）→ 1000题A章节联动。
- 英语一：单词一轮到二轮 → 阅读真题 2009-2020 → 小三门导入（翻译/新题型/完型）→ 作文模板初版。
- 政治：马原/毛概/史纲/思修框架铺底 + 肖1000入门。
- 专业课843：技术线（数据结构/算法/网络/OS/硬件/概率统计）与设计线（洞察/思维/表现/手绘/方案）并行。

### 3.2 强化期（7/1-9/30）

- 数学二：专题强化 + 1000题B + 真题 2009-2024 限时分段训练。
- 英语一：阅读二刷 + 小三门专项 + 作文方法论与限时训练。
- 政治：肖1000二刷 + 背诵手册 + 时政。
- 专业课843：技术综合题与设计综合题同步提速。

### 3.3 冲刺期（10/1-12/20）

- 数学二：真题近年套卷 + 合工大/李艳芳/欧几里得/氮云模拟 + 错题本三刷。
- 英语一：真题三刷 + 作文模板定稿 + 每日保温。
- 政治：肖八完成后转肖四，主观题背诵至少三轮。
- 专业课843：技术套题与设计套题交替，保留手绘快题频率。
