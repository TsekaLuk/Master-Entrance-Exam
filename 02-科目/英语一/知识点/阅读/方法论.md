# 阅读理解方法论（唐迟）

> 阅读占40分，是英语最重要的模块

## 做题流程

```
1. 先读题目（题干+选项，不需全读）
2. 带着问题读文章（定位）
3. 定位到原文区域，找同义替换
4. 排除明显错误选项
```

## 五大题型

### 主旨题
- 串读首尾段 + 各段主题句
- 正确答案：概括全文，不以偏概全
- 排除：过于具体/范围过大

### 细节题
- 定位词：专有名词/数字/特殊标点/转折词
- 同义替换识别（考点核心）
- 排除：未提到/反向描述/混淆信息

### 例证题
- 例子是为了支撑论点
- 答案找例子所在**段落的主题句**（不是例子本身）

### 态度题
- 找情感词（positive/negative/neutral）
- 作者态度 ≠ 文中人物态度
- 常见正确态度：critical, skeptical, objective

### 推理题
- 答案 = 文章隐含信息，需推导
- 不能直接在文中找到，但必须有依据

## 长难句处理

1. 找主干（主谓宾）
2. 识别从句类型（定语/状语/名词）
3. 还原修饰关系
