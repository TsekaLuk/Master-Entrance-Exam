## 操作系统的功能

| 功能 | 说明 |
|------|------|
| 进程管理 | 进程调度、并发控制、死锁处理 |
| 内存管理 | 分配/回收内存、虚拟内存、页面置换 |
| 文件管理 | 文件系统结构、目录管理、权限控制 |
| 设备管理 | I/O设备调度、驱动程序管理 |

---

## 进程与线程

| 概念 | 说明 |
|------|------|
| 进程 | 程序的一次执行，资源分配基本单位 |
| 线程 | 进程内的执行单元，CPU调度基本单位 |
| 区别 | 进程有独立地址空间，线程共享进程资源 |

### 进程状态转换

```
新建 → 就绪 ⇌ 运行 → 终止
              ↕
            阻塞
```

### 进程调度算法

| 算法 | 特点 |
|------|------|
| FCFS（先来先服务）| 简单，可能"护航效应" |
| SJF（短作业优先）| 平均等待时间最短，可能饥饿 |
| 时间片轮转 | 公平，响应时间有保障 |
| 优先级调度 | 灵活，可能低优先级饥饿 |

## 内存管理

### 分页与分段

| 方式 | 单位 | 特点 |
|------|------|------|
| 分页 | 固定大小的页 | 无外部碎片，有内部碎片 |
| 分段 | 可变大小的段 | 按逻辑分段，有外部碎片 |
| 段页式 | 段+页 | 结合两者优点 |

### 页面置换算法

| 算法 | 策略 | 说明 |
|------|------|------|
| FIFO | 淘汰最早进入的页 | 可能Belady异常 |
| LRU | 淘汰最近最少使用的页 | 接近最优，实现复杂 |
| OPT | 淘汰未来最长时间不用的页 | 理论最优，无法实现 |

## 死锁

**必要条件（四个同时满足才会死锁）：**
1. 互斥条件
2. 请求与保持
3. 不可剥夺
4. 循环等待

**银行家算法**：预防死锁，判断资源分配是否安全
