# 硬件组成原理

> 843考纲模块 A2 | 关联：[[01-数制与编码]] · [[03-指令系统与运算]]

---

## 一、计算机硬件五大部件

```
        ┌─────────────┐
        │    控制器    │  ← 指挥中心
        │    运算器    │  ← ALU（算术逻辑单元）
        └──────┬──────┘
               │ CPU（控制器 + 运算器 + 寄存器）
        ┌──────┼──────┐
        ↓      ↓      ↓
     存储器   输入   输出
```

| 部件 | 功能 | 典型设备 |
|------|------|---------|
| **运算器（ALU）** | 算术/逻辑运算 | CPU核心 |
| **控制器** | 取指、译码、执行 | CPU核心 |
| **存储器** | 存放程序和数据 | RAM、ROM、硬盘 |
| **输入设备** | 向计算机输入信息 | 键盘、鼠标 |
| **输出设备** | 输出计算结果 | 显示器、打印机 |

---

## 二、CPU 内部结构

| 组件 | 作用 |
|------|------|
| **ALU**（算术逻辑单元）| 加/减/与/或/移位运算 |
| **寄存器** | 高速临时存储（PC、IR、ACC、MAR、MDR）|
| **控制单元（CU）** | 产生控制信号，协调各部件 |
| **内部总线** | 连接ALU、寄存器和CU |

**关键寄存器**：
- **PC**（程序计数器）：存放下一条指令地址，自动递增
- **IR**（指令寄存器）：存放当前正在执行的指令
- **MAR**（内存地址寄存器）：访问内存时存放地址
- **MDR**（内存数据寄存器）：存放从内存读出或要写入的数据

---

## 三、存储器层次结构

```
速度快↑ 容量小↑ 成本高↑
─────────────────────────
         寄存器
         高速缓存 Cache
         主存（RAM）
         磁盘（HDD/SSD）
         光盘/磁带
─────────────────────────
速度慢↓ 容量大↓ 成本低↓
```

| 类型 | 特征 | 断电 |
|------|------|------|
| **RAM**（随机存储器）| 可读写，主存 | 丢失 |
| **ROM**（只读存储器）| 只读，存BIOS | 保留 |
| **Cache**（高速缓存）| 存CPU近期用到的数据 | 丢失 |
| **虚拟内存** | 用磁盘空间扩展主存 | — |

> **二级结论**：Cache命中率越高，CPU等待越少；局部性原理（时间局部性/空间局部性）是Cache有效的前提

---

## 四、总线

**定义**：连接各部件的公共信息传输线

| 总线类型 | 传输内容 |
|---------|---------|
| **数据总线（DB）** | 数据，双向 |
| **地址总线（AB）** | 地址，单向（CPU→存储器）|
| **控制总线（CB）** | 控制信号，双向 |

> **二级结论**：地址总线位数决定**最大寻址空间**（$2^n$ 字节）；数据总线位数决定一次传输的**数据宽度**（即字长）

---

## 五、IO（输入输出）方式

| 方式 | 特点 |
|------|------|
| **程序查询（轮询）** | CPU主动轮询设备状态，效率低 |
| **中断方式** | 设备完成后中断CPU，效率较高 |
| **DMA（直接内存访问）** | 数据在内存和设备间直接传输，不经过CPU，速度最快 |

---

## 公式快查

| 概念 | 要点 |
|------|------|
| 地址总线 $n$ 位 | 寻址空间 $2^n$ 字节 |
| Cache 命中率高 | 平均访问时间接近 Cache 速度 |
| DMA | 不占用CPU，适合大批量数据传输 |
