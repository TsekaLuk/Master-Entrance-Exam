## 特色内容

### 数学二知识图谱
- **高数**：极限 → 导数 → 积分 → 微分方程 → 多元函数，18个知识点文件
- **线代**：行列式 → 矩阵 → 向量 → 方程组 → 特征值 → 二次型
- 每个文件包含：核心公式 + **二级结论**（推论和易错点）+ 双链导航

### 843 专业课框架（官方大纲对齐）
- **A1 数理基础**：概率论（条件概率/贝叶斯/随机变量）+ 数理统计（MLE/区间估计）
- **A2 计算机硬件基础**：CPU结构/流水线/存储层次/总线/IO
- **A3 数据结构**：线性结构/树与二叉树/排序算法/查找算法
- **A4 网络软硬件技术**：OSI模型/TCP协议/HTTP/DNS/组网

### 英语一词根词缀双链图谱
- 来源：GRE核心词汇（1036词·35主题）
- **58个核心词根**：每个词根文件列出所有衍生词 + 主题回链
- **38个高频前缀**：从含义推导词义的逻辑链
- **61个常用后缀**：词性转换规律
- 口诀：**前缀定方向，词根定意义，后缀定词性**

## 如何使用

1. 安装 [Obsidian](https://obsidian.md)（免费）
2. 克隆本仓库：`git clone https://github.com/TsekaLuk/Master-Entrance-Exam.git`
3. 在 Obsidian 中打开文件夹作为 Vault
4. 推荐插件：**Dataview** · **Templater** · Calendar · Graph View

详见 [SETUP.md](SETUP.md) 获取完整初始化指南（含插件安装、框架说明、GRE词汇内容说明）。
