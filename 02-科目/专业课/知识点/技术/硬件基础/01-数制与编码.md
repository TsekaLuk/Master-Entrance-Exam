# 数制与编码

## 进制转换

| 进制 | 基数 | 数字范围 |
|------|------|----------|
| 二进制 | 2 | 0,1 |
| 八进制 | 8 | 0-7 |
| 十进制 | 10 | 0-9 |
| 十六进制 | 16 | 0-9, A-F |

**十进制 → 二进制：** 除2取余法（从下往上读）
**二进制 → 十进制：** 按权展开相加

## 字符编码

| 编码 | 说明 |
|------|------|
| ASCII | 128个字符，7位，英文字符 |
| Unicode | 统一字符集，支持全球文字 |
| UTF-8 | Unicode的可变长编码（1-4字节），兼容ASCII |
| GBK | 中文扩展编码（2字节/汉字）|

## 计算机硬件组成

| 部件 | 功能 |
|------|------|
| CPU | 运算+控制，包含ALU+控制器+寄存器 |
| RAM | 主存，断电丢失 |
| ROM | 只读存储（BIOS）|
| 硬盘 | 外存，永久存储 |
| 总线 | 数据/地址/控制信号传输通道 |
