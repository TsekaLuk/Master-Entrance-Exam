## 五、考试大纲

### 信息技术类（50%）

1. **数理基础**
   - 概率论：随机事件与概率、常见随机变量及分布、随机变量的数字特征
   - 数理统计：总体与样本、统计量

2. **计算机硬件基础**
   - 数值表示（二进制、八进制、十六进制）
   - 数字与字符编码
   - 计算机硬件组成

3. **数据结构**
   - 基本概念
   - 线性与非线性结构
   - 查找与排序

4. **网络软硬件技术**
   - 网络基础、协议
   - 网络组建及应用
   - Internet网

### 创新设计类（50%）

1. **洞察力**：发现、思考与分析现实社会问题
2. **学习力**：知识吸收、记忆、消化与运用
3. **思维力**：运用互联网+思维提出创新解决方案
4. **表现力**：设计表现技能（手绘、软件绘图）
