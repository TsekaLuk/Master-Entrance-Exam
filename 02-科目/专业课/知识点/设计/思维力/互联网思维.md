# 互联网思维

## 核心特征

| 思维 | 核心概念 | 典型案例 |
|------|----------|----------|
| 用户中心思维 | 一切以用户价值为出发点 | 微信"以用户为中心"的产品哲学 |
| 数据驱动思维 | 用数据说话，量化决策 | A/B测试、漏斗分析 |
| 迭代思维 | 快速试错，持续迭代 | MVP→小步快跑 |
| 平台思维 | 构建生态，连接多方 | 淘宝、微信生态 |
| 跨界思维 | 打破行业边界 | 互联网+传统产业 |

## 互联网产品设计原则

1. **简单**：功能聚焦，减少认知负担
2. **快速**：极速响应，降低等待焦虑
3. **个性化**：算法推荐，千人千面
4. **社交化**：分享、互动、UGC
5. **免费+增值**：基础免费，高级付费

## 产品分析框架（案例题常用）

```
用户是谁（Who）→ 核心需求（What）→ 使用场景（When/Where）
→ 解决方案（How）→ 商业模式（Business Model）
```
