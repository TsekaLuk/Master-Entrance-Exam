# 大作文九宫格方法（潘老师）

> 大作文 20分，图画作文为主 | 九宫格法在题目变化时更灵活

## 三段结构

### 第一段：描述图画

**模板：**
> As is vividly depicted in the picture, [描述图画内容]. The cartoon aims to convey the message that [主题].

### 第二段：分析原因 / 阐述意义

**原因分析（二选一）：**
> There are several factors accounting for this phenomenon. To begin with, [原因1]. Moreover, [原因2]. Last but not least, [原因3].

**意义阐述：**
> The significance of [主题] cannot be overstated. In the first place, ... In addition, ...

### 第三段：总结与建议

**模板：**
> In conclusion, [总结]. It is high time that we [建议/行动]. Only in this way can we [期望结果].

## 九宫格思路展开

| | 社会现象类 | 对比图类 | 时间变化类 |
|-|-----------|---------|-----------|
| 描述 | 图中...在做... | 左图...右图... | 过去...现在... |
| 原因 | 经济/文化/科技 | 分析对比原因 | 发展变化原因 |
| 展望 | 个人/社会建议 | 启示 | 趋势预测 |

## 高分句型

- Not only ... but also ...
- It is ... that ... （强调句）
- The more ... the more ...
- Only by + doing ... can we ...
- What is particularly striking is that ...
