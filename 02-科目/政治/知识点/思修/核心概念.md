# 思想道德与法治

> 考研政治思修分值约 16分 | 选择题为主，少量分析题 | 近年与时政结合紧密

## 一、人生观与价值观

### 人生目的（人生观的核心）

| 错误人生观 | 正确人生观 |
|-----------|-----------|
| 享乐主义（及时行乐）| 为人民服务，实现人生价值 |
| 拜金主义（金钱万能）| 追求物质与精神的统一 |
| 个人主义 | 个人与社会统一 |

### 人生价值的评价

- **社会价值**（贡献）> **自我价值**（索取）
- 评价标准：是否对社会、国家、人民有贡献
- 实现路径：实践（劳动是创造价值的根本途径）

## 二、理想信念

### 理想的层次

```
最崇高的社会理想：实现共产主义（最高理想）
    ↕ 辩证统一
现实的奋斗目标：全面建设社会主义现代化国家（共同理想）
```

**个人理想与社会理想的关系**：
- 社会理想决定和制约个人理想
- 个人理想是社会理想的具体体现

### 坚定理想信念的意义

1. 精神支柱（提供精神动力）
2. 行动向导（指明正确方向）
3. 精神纽带（凝聚社会合力）

## 三、爱国主义

### 新时代爱国主义

**爱国主义 = 爱国 + 爱党 + 爱社会主义的有机统一**（重要！）
- 新时代不能把爱国主义与爱党、爱社会主义割裂

### 爱国主义与经济全球化

- 全球化不是否定爱国主义的理由
- 经济全球化条件下更需要弘扬爱国主义精神
- 爱国主义与世界主义不矛盾（国际主义）

## 五、道德

### 道德的本质
- 道德是社会关系的产物，由社会经济关系（生产方式）决定
- 道德通过社会舆论、风俗习惯、内心信念发挥作用（区别于法律的强制性）

### 道德的四个领域

| 领域 | 核心要求 |
|------|----------|
| 社会公德 | 文明礼貌、助人为乐、爱护公物、保护环境、遵纪守法 |
| 职业道德 | 爱岗敬业、诚实守信、办事公道、服务群众、奉献社会 |
| 家庭美德 | 尊老爱幼、男女平等、夫妻和睦、勤俭持家、邻里团结 |
| 个人品德 | 自尊自爱、自律自强、宽厚善良、积极向上 |

### 中华传统美德

- **仁**：仁者爱人，己所不欲勿施于人
- **义**：舍生取义，见义勇为
- **礼**：克己复礼，谦恭有礼
- **智**：格物致知，明辨是非
- **信**：诚实守信，言而有信

## 六、法治（重点！）

### 全面依法治国

**总目标**：建设中国特色社会主义法治体系，建设社会主义法治国家

**四大任务**：
1. 完善以宪法为核心的中国特色社会主义法律体系
2. 建立高效的法治实施体系
3. 建设严密的法治监督体系
4. 建立有力的法治保障体系

### 法治思维

**法治与德治的关系**：
- 法治以其权威性和强制性规范社会行为
- 德治以其说服力和感召力提高社会成员道德素质
- 二者相辅相成，缺一不可

### 宪法

- 宪法是国家根本法，具有最高法律效力
- 宪法是其他法律的立法基础和依据
- 公民既有权利（宪法赋予）也有义务（宪法规定）

### 权利与义务

- 权利与义务在法律关系上相互对应
- 享有权利同时须履行义务
- 权利可以放弃，但义务不能拒绝

## 高频易错点

- ❌ "法律是万能的，道德可以用法律代替" ← 道德和法律各有其调整范围，相互补充
- ❌ "爱国主义就是狭隘民族主义" ← 真正的爱国主义包含开放包容
- ❌ "理想只要崇高就好，不必考虑现实" ← 理想要与现实相结合
