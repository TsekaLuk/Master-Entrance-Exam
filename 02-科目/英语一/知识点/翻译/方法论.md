# 英译汉方法论（唐静）

> 翻译 10分，共5句，每句2分

## 基本原则

1. **信**：忠实原文，不增不减
2. **达**：表达流畅，符合中文习惯
3. **雅**：（考研级别达到信、达即可）

## 解题步骤

```
1. 通读全句，理解大意
2. 识别句子主干（主谓宾）
3. 识别从句类型（定语/状语/名词从句）
4. 逐词/词组翻译
5. 调整语序，使中文通顺
```

## 常见句式处理

| 句式 | 处理方法 |
|------|----------|
| 定语从句 | 后置→前置修饰，长从句用"的"或单独成句 |
| 状语从句 | 时间/原因/结果/条件从句，调整到合适位置 |
| 被动语态 | 多数转为主动，或用"被/由/受" |
| 否定结构 | 注意部分否定 (not all ≠ none) |
| 形式主语 | It is ... that ... → "...是..." |
