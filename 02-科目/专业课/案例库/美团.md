| 项目 | 内容 |
|------|------|
| 产品定位 | 本地生活服务平台（O2O聚合） |
| 核心业务 | 外卖 / 酒旅 / 到店餐饮 / 闪购 |

---

## 四维分析

### 1. 洞察力 — 发现了什么问题？

> 核心洞察：

- **线下商家**：获客成本高、数字化工具缺失
- **用户**：餐厅信息分散、排队等候时间长
- **市场**：本地生活服务高度分散，无统一入口

### 2. 思维力 — 如何解决问题？

**设计思维应用：**

| 阶段 | 美团的做法 |
|------|----------|
| 同理心 | 深入走访商家，理解运营痛点 |
| 定义问题 | 商家数字化 + 用户效率提升 |
| 创意发散 | 团购切入 → 外卖 → 到店 → 超级平台 |
| 原型验证 | 城市地推模式快速扩张验证 |
| 测试迭代 | 千城大战，胜者为王 |

**互联网思维：**
- 用户中心：蜂鸟即时配送，30分钟送达
- 数据驱动：骑手调度算法、商家评分体系
- 快速迭代：从团购到外卖的业务转型

### 3. 学习力 — 跨界融合了什么？

- 借鉴 Amazon 的履约配送体系 → 自建蜂鸟配送
- 借鉴 Yelp 的点评模式 → 与大众点评合并
- 金融：美团月付 / 商家贷款（SaaS工具变现）

### 4. 表现力 — 可视化草图

```
[ 用户 ] ──点餐──→ [ 美团App ] ──派单──→ [ 商家 ]
                         ↓
                   [ 蜂鸟骑手 ]
                         ↓
                   [ 30min送达 ]
```

---

## 答题关键词

- **O2O闭环**：线上交易 + 线下履约
- **供给侧数字化**：美团给商家提供收银/进货/SaaS工具
- **即时零售**：从外卖延伸到闪购（商超/药店）
- **本地生活**：高频（外卖）带低频（酒旅、婚庆）
