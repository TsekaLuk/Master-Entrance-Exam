# 数学错题汇总

> 错题是最宝贵的资源。每次做错都是一次暴露弱点的机会。
> 使用 Templater 模板 `错题记录` 新建每道错题，本页自动聚合。

---

## 错题统计看板

```dataviewjs
const pages = dv.pages('"02-科目/数学/错题"')
  .where(p => p.错误类型 && p.file.name !== "错题汇总");

const byType = {};
const byChapter = {};
const pending = [];
const today = dv.date("today");

for (const p of pages) {
  // 按错误类型统计
  const t = p.错误类型 || "未分类";
  byType[t] = (byType[t] || 0) + 1;
  // 按章节统计
  const c = p.章节 || "未分类";
  byChapter[c] = (byChapter[c] || 0) + 1;
  // 待复习
  if (p["next-review"] && dv.date(p["next-review"]) <= today) {
    pending.push(p);
  }
}

dv.header(3, `共 ${pages.length} 道错题`);

// 错误类型分布
dv.header(4, "错误类型分布");
dv.table(["类型", "数量"],
  Object.entries(byType).sort((a,b) => b[1]-a[1])
);

// 高频章节
dv.header(4, "高频出错章节");
dv.table(["章节", "次数"],
  Object.entries(byChapter).sort((a,b) => b[1]-a[1]).slice(0,8)
);

// 待复习
if (pending.length > 0) {
  dv.header(4, `⏰ 今日待复习 (${pending.length} 道)`);
  dv.table(["题目", "章节", "上次日期"],
    pending.map(p => [p.file.link, p.章节, p.date])
  );
}
```

## 全部错题列表

```dataviewjs
const pages = dv.pages('"02-科目/数学/错题"')
  .where(p => p.错误类型)
  .sort(p => p.date, "desc");

dv.table(
  ["题目", "章节", "错误类型", "来源", "掌握状态", "下次复习"],
  pages.map(p => [
    p.file.link,
    p.章节 || "—",
    p.错误类型 || "—",
    p.来源 || "—",
    p.掌握状态 || "待攻克",
    p["next-review"] || "—"
  ])
);
```
