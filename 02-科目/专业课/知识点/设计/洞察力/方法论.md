# 洞察力

> 发现问题 → 分析问题 → 提炼本质

## 三个层次

| 层次 | 能力 | 方法 |
|------|------|------|
| 发现问题 | 观察社会现象，识别痛点 | 用户访谈、日志法、影子跟踪 |
| 分析问题 | 归因分析、多维度思考 | 5W2H、鱼骨图、How Might We |
| 提炼本质 | 从现象到本质的抽象 | Why×5、洞见陈述 |

## 用户研究方法

| 方法 | 类型 | 适用场景 |
|------|------|----------|
| 深度访谈 | 定性 | 发现深层需求 |
| 问卷调查 | 定量 | 验证假设 |
| 观察法/影子跟踪 | 定性 | 了解真实行为 |
| 焦点小组 | 定性 | 探索态度观点 |
| 日记研究 | 定性 | 长期行为追踪 |

## 洞见陈述模板

```
[用户类型] 需要 [需求/目标]，因为 [洞见/原因]，
但目前存在 [障碍/痛点]，因此我们相信 [设计机会]。
```
