# 计算机网络 — 知识地图 (MOC)

## 分值说明

408 计算机网络共 **25分**：
- 选择题：约 **16分**（每题2分，约8题）
- 综合应用题：约 **9分**（1~2道大题）

复习优先级：TCP/IP >> 路由算法 >> 数据链路层 >> 应用层协议

---

## 五层模型总览

```
┌─────────────────────────────────────┐
│  应用层 Application Layer           │  HTTP/HTTPS, DNS, FTP, SMTP, DHCP
├─────────────────────────────────────┤
│  传输层 Transport Layer             │  TCP, UDP
├─────────────────────────────────────┤
│  网络层 Network Layer              │  IP, ICMP, ARP, RIP, OSPF, BGP
├─────────────────────────────────────┤
│  数据链路层 Data Link Layer         │  以太网, PPP, CSMA/CD, MAC
├─────────────────────────────────────┤
│  物理层 Physical Layer              │  信号编码, 奈奎斯特, 香农, 传输介质
└─────────────────────────────────────┘
```

> 注意：OSI 七层模型在物理层和数据链路层之间无差异，在应用层上方增加表示层和会话层。408 考试以 **五层模型** 为主。
